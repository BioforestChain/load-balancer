# Load Balancer

Load balancing using Golang. [source code](./binary/)

> 使用 Golang 实现的负载均衡。[源码](./binary/)

## Usage

```ts
import { LoadBalancer } from "@bfchain/load-balancer";

const lb = new LoadBalancer();
const res = await lb.start({
  scheme: "http",
  host: "0.0.0.0",
  port: 8888,
  servers: [
    {
      name: "Server A",
      scheme: "http",
      host: "127.0.0.1",
      port: 9800,
    },
    {
      name: "Server B",
      scheme: "http",
      host: "127.0.0.1",
      port: 9801,
    },
  ],
  // custom static route
  routes: [
    {
      mode: ":path",
      value: "/a/*",
      to: ["Server A"],
    },
    {
      mode: ":path",
      value: "/a/*",
      to: ["Server B"],
    },
  ],
});
```

### Route Mode

- `:path`
  ```json
  {
    "mode": ":path",
    "value": "/a/*",
    "to": ["Server A"]
  }
  ```
- `:method`
  ```json
  {
    "mode": ":method",
    "value": "GET",
    "to": ["Server A"]
  }
  ```
- `:header`
  ```json
  {
    "mode": ":header",
    "key": "set-cookie",
    "value": "x=x",
    "to": ["Server A"]
  }
  ```
- `:and`
  ```json
  {
    "mode": ":and",
    "left": {
      "mode": ":path",
      "value": "/a/*"
    },
    "right": {
      "mode": ":path",
      "value": "/a/*"
    },
    "to": ["Server A"]
  }
  ```

## Run Load-Balancer without nodejs 独立运行程序

```bash
./load-balancer -config ./config.json
```
